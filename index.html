FAQ

Ответы на частые вопросы

Нужно ли мне покупать свои прокси?

Да, бот поддерживает работу через HTTP и SOCKS5 прокси. Ротация происходит автоматически.

Безопасно ли это для аккаунтов?

Да, встроен 4-фазный прогрев и эмуляция паттернов поведения реального человека (паузы, скроллинг).

Где работает софт?

Приложение поддерживается на ОС Windows 10/11 (64-bit). Вы также можете устанавливать его на выделенный сервер/дедик для работы 24/7.

Какую ИИ-модель использует бот?

Вы можете самостоятельно вставить свой API ключ в настройках (например, Groq или OpenAI). Пользователь сам выбирает лучшую модель для задачи.
